# 자료구조란?

* 용어: 자료구조, 데이터 구조, data structure
* 대량의 데이터를 효율적으로 관리할 수 있는 데이터 구조를 의미
* 코드 상에서 효율적으로 데이터를 처리하기 위해, 데이터 특성에 따라, 체계적으로 데이터를 구조화 해야함
    * 어떤 데이터 구조를 사용하느냐에 따라, 코드 효율이 달라짐 

## 대표적인 자료구조

* 배열, 스택, 큐, 링크드 리스트, 해쉬 테이블, 힙 등
