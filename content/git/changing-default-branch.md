## 기본 브랜치 변경하기

1. 먼저 `git checkout -b 새로운 브랜치명`으로 브랜치를 생성합니다.
2. `Settings`에 들어가서 `Branches` 메뉴를 클릭합니다.
3. `Default branch`를 새로만든 브랜치로 바꾼 후 `Update` 버튼을 클릭하고 경고를 읽은 다음 `I understand ~` 버튼을 누르면 됩니다.
